Acerca de este Proyecto

Streamflix es una aplicación de streaming para Android (móvil y TV) con una arquitectura muy inteligente y, lo más importante para nosotros, modular. La clave de todo el proyecto reside en la separación total entre la interfaz de usuario (lo que el usuario ve) y las fuentes de contenido. Estas fuentes de contenido son lo que llamamos "Providers" (Proveedores), y ahí es donde entra nuestro trabajo.
